# Rastgele tavsiye notları

``https://api.adviceslip.com/advice`` api adresinden ``fetch()`` metodunu kullanarak tavsiye notlarının getirilmektedir.

Sayfa yüklenirken advice getirilmesi için ``getAdvice()`` metodu kullanılmıştır.

New advice butonunun click event'ine de ``getAdvice()`` metodu atanmıştır.
